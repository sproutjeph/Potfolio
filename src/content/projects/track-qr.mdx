## What it is

TrackQR is a dynamic QR platform. Unlike a static QR (which encodes a fixed URL at print time), every TrackQR code is a redirect you can rewrite at any time — so the same poster, sticker, or business card stays useful as the destination evolves.

## What it does

- **Dynamic codes** — change the destination URL without reprinting
- **Branded codes** — embed logo, colour, frame, custom shape
- **Analytics** — scan count, geo-IP, device, browser, time-of-day, referrer
- **Bulk generation + CSV export** for campaigns
- **Team workspaces** with role-based access

## Stack

Next.js + TypeScript on the front. A Bun-powered API service for the redirect path (sub-50ms TTFB on 1M+ scans). MongoDB Atlas for storage. Fly.io for global edge deployment so scans resolve from the nearest region.

## Outcomes

500+ brands, 10,000+ codes in circulation, 1M+ total scans served.
